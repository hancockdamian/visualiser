# visualiser

## setup

```bash
git clone git@github.com:hancockdamian/visualiser.git
cd visualiser
```

### start a development server:

```bash
npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```

### build prod

```bash
npm run build
```

### preview production build

```bash
npm run preview
```
